FreeSpace – Pay-Per-Minute Space Rental SaaS (Laravel 12 + Inertia.js + Vue.js)

Turn idle corners of shops, supermarkets, or offices into revenue. FreeSpace lets Shop Owners set a per-minute price; customers simply scan a QR to start/stop usage and pay for the exact minutes they use. Perfect for a SaaS platform where you earn an admin commission.

Why Choose FreeSpace?

  • SaaS-ready: Complete frontend, backend & REST APIs — rebrand and launch quickly.
  • Built-in business model: Per-minute pricing with admin commission & settlement workflow.
  • Realtime UX: Live start/stop & notifications via Laravel Reverb + Laravel notifications table.
  • Clear roles: Customer, Shop Owner, Admin, Super Admin using spatie/laravel-permission.
  • Developer-friendly: Service layer + DI, Form Requests, API Resources, clean routes.
  • Mobile/SPA ready: All APIs documented with Scribe — build native apps or a separate SPA anytime.
  • Ops ready: Dockerized app, Kubernetes manifests, environment templates & sample Nginx/PHP-FPM config.
How It Works
  1. Users register and receive a personal QR card.
  2. At a registered shop, scan to start the timer.
  3. Scan again to stop; pay instantly or keep as due.
Example: Rate = 10 BDT/min, Duration = 20 min → Total 200 BDT. With 10% admin fee, Admin gets 20 BDT, Shop Owner gets 180 BDT.

Core Modules & Features

Installation

Single Platform (recommended)

  1. Upload files and create database.
  2. Copy environment template (.env.example or .env.production) and update values.
  3. Install dependencies: composer install, npm ci && npm run build.
  4. Run migrations & seeders; set queue/cron as needed.
  5. Configure Reverb/WebSocket and mail/SMS gateways (optional).

Docker / Kubernetes

  • Docker: docker-compose.yml for dev/prod (with .env.docker).
  • Kubernetes: Apply manifests in k8s/ for cluster deployment.

Business Opportunity

Launch a marketplace-style SaaS: onboard multiple Shop Owners, set platform-wide or shop-wise commissions, run campaigns, and earn from every minute billed. The APIs and documentation make it easy to add a mobile app or a separate SPA later.

FAQ

Does it include full source code?
Yes — Laravel + Inertia/Vue source with docs.

Is payment gateway included?
Yes — Stripe Checkout is integrated out of the box for online payments (alongside cash flow & settlement logic). A Stripe Webhook is implemented to verify payments server-side and automatically update transactions/settlements. You can extend or swap gateways later using the same hooks/endpoints.

Is it multi-role?
Yes — Customer, Shop Owner, Admin, Super Admin with spatie/laravel-permission.

Can I build mobile apps later?
Absolutely — complete REST APIs with Scribe docs are included.
